\doxysubsection{Detailed Description}
Trieda zapúzdrujúca ovládanie FM rádia \doxylink{class_si4703}{Si4703}. 

Trieda rieši\+:
\begin{DoxyItemize}
\item inicializáciu a zapnutie/vypnutie čipu,
\item nastavenie pásma, rozstupu kanálov a de-\/emfázy,
\item ladenie konkrétnej frekvencie a seekovanie nahor/nadol,
\item získanie RSSI, informácií o stereo/mono režime, mute stave a pod.,
\item ovládanie hlasitosti a rozšíreného rozsahu hlasitosti,
\item čítanie RDS dát,
\item konfiguráciu GPIO pinov \doxylink{class_si4703}{Si4703}.
\end{DoxyItemize}

Komunikácia prebieha cez I2C (2-\/wire režim) alebo 3-\/wire rozhranie podľa interných funkcií triedy.
