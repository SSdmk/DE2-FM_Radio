\doxysubsection{Detailed Description}
Deklarácie UI udalostí a rozhrania pre obsluhu tlačidiel a enkódera rádia. 

Tento modul definuje\+:
\begin{DoxyItemize}
\item typy udalostí z používateľského rozhrania (tlačidlá, enkóder),
\item režimy práce enkódera (hlasitosť, ladenie),
\item API funkcie na inicializáciu a spracovanie udalostí,
\item pomocnú funkciu na zistenie ON/\+OFF stavu rádia z pohľadu UI vrstvy. 
\end{DoxyItemize}

Spracuje jednu udalosť z tlačidiel alebo enkódera. 

Túto funkciu volaj vždy, keď vstupná vrstva (Button/\+Rotary\+Encoder) deteguje niektorý z typov \doxylink{button__function_8h_a9a99e204659fdc9573b3dc140ffeedd3}{ui\+\_\+event\+\_\+t} a chceš ho premietnuť do akcie v UI (zmena hlasitosti, ladenia, prepínanie režimov, mute, power on/off a pod.).

Typické použitie v hlavnej slučke\+:
\begin{DoxyItemize}
\item prečítať stav tlačidiel a enkódera,
\item preložiť ho na \doxylink{button__function_8h_a9a99e204659fdc9573b3dc140ffeedd3}{ui\+\_\+event\+\_\+t},
\item odovzdať sem na spracovanie.
\end{DoxyItemize}

Spracuje jednu udalosť z tlačidiel alebo enkódera.

Podľa hodnoty {\ttfamily ev} vykoná\+:
\begin{DoxyItemize}
\item krok seeku hore/dole (ľavé/pravé tlačidlo),
\item naladenie/uloženie obľúbenej frekvencie (horné tlačidlo),
\item prepnutie mute, resp. zapnutie/vypnutie rádia (dolné tlačidlo),
\item zmenu hlasitosti alebo frekvencie (otáčanie enkódera),
\item prepnutie režimu enkódera VOLUME/\+TUNE (klik na enkóder).
\end{DoxyItemize}
