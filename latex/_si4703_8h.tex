\doxysubsection{Detailed Description}
Rozhranie pre ovládanie FM rádia \doxylink{class_si4703}{Si4703}. 

Tento hlavičkový súbor obsahuje\+:
\begin{DoxyItemize}
\item konštanty a symbolické názvy pre nastavenie rádia (pásmo, de-\/emfáza, rozstup kanálov, GPIO režimy, parametre seeku atď.),
\item deklaráciu triedy \doxylink{class_si4703}{Si4703}, ktorá zapúzdruje komunikáciu s čipom,
\item dátové štruktúry (union + bitové polia) reprezentujúce registre \doxylink{class_si4703}{Si4703} vrátane „shadow“ kópie všetkých registrov.
\end{DoxyItemize}

Trieda \doxylink{class_si4703}{Si4703} poskytuje metódy na\+:
\begin{DoxyItemize}
\item zapnutie/vypnutie čipu,
\item ladenie frekvencie a seekovanie,
\item čítanie RSSI a stavových príznakov (stereo, RDS, STC),
\item nastavenie hlasitosti, mute, mono, rozšíreného rozsahu hlasitosti,
\item prácu s RDS a GPIO pinmi. 
\end{DoxyItemize}
