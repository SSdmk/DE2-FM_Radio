\doxysubsection{Detailed Description}
Implementácia obsluhy UI udalostí (tlačidlá + enkóder) pre FM rádio. 

Tento modul\+:
\begin{DoxyItemize}
\item drží interný stav rádia (zapnuté/vypnuté, obľúbená stanica, režim enkódera),
\item mapuje udalosti z UI ({\ttfamily \doxylink{button__function_8h_a9a99e204659fdc9573b3dc140ffeedd3}{ui\+\_\+event\+\_\+t}}) na konkrétne akcie nad objektom {\ttfamily \doxylink{class_si4703}{Si4703}},
\item pri niektorých akciách aktualizuje OLED (napr. uloženie obľúbenej stanice). 
\end{DoxyItemize}

Spracuje jednu udalosť z tlačidiel alebo enkódera.

Podľa hodnoty {\ttfamily ev} vykoná\+:
\begin{DoxyItemize}
\item krok seeku hore/dole (ľavé/pravé tlačidlo),
\item naladenie/uloženie obľúbenej frekvencie (horné tlačidlo),
\item prepnutie mute, resp. zapnutie/vypnutie rádia (dolné tlačidlo),
\item zmenu hlasitosti alebo frekvencie (otáčanie enkódera),
\item prepnutie režimu enkódera VOLUME/\+TUNE (klik na enkóder).
\end{DoxyItemize}
