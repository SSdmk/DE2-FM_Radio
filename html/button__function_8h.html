Detailed Description

Deklarácie UI udalostí a rozhrania pre obsluhu tlačidiel a enkódera rádia.

Tento modul definuje:

  • typy udalostí z používateľského rozhrania (tlačidlá, enkóder),
  • režimy práce enkódera (hlasitosť, ladenie),
  • API funkcie na inicializáciu a spracovanie udalostí,
  • pomocnú funkciu na zistenie ON/OFF stavu rádia z pohľadu UI vrstvy.

◆ radio_ui_handle_event()

void radio_ui_handle_event ( ui_event_t ev)

Spracuje jednu udalosť z tlačidiel alebo enkódera.

Túto funkciu volaj vždy, keď vstupná vrstva (Button/RotaryEncoder) deteguje niektorý z typov ui_event_t a chceš ho premietnuť do akcie v UI (zmena hlasitosti, ladenia, prepínanie režimov, mute, power on/off a pod.).

Typické použitie v hlavnej slučke:

  • prečítať stav tlačidiel a enkódera,
  • preložiť ho na ui_event_t,
  • odovzdať sem na spracovanie.
Parameters
evUdalosť z používateľského rozhrania, ktorú chceš spracovať.

Spracuje jednu udalosť z tlačidiel alebo enkódera.

Podľa hodnoty ev vykoná:

  • krok seeku hore/dole (ľavé/pravé tlačidlo),
  • naladenie/uloženie obľúbenej frekvencie (horné tlačidlo),
  • prepnutie mute, resp. zapnutie/vypnutie rádia (dolné tlačidlo),
  • zmenu hlasitosti alebo frekvencie (otáčanie enkódera),
  • prepnutie režimu enkódera VOLUME/TUNE (klik na enkóder).

Funkcia predstavuje čistú UI logiku nad globálnym objektom radio (Si4703 tuner) a OLED displejom.

Parameters
evUdalosť prijatá z vrstvy vstupu (tlačidlá/enkóder).

◆ radio_ui_is_on()

bool radio_ui_is_on ( void )

Zistí, či je rádio podľa UI logiky aktuálne zapnuté.

Funkcia nečítá priamo HW stav, ale vracia vnútorný stav, ktorý udržiava UI vrstva (napr. po volaní radio.powerUp() / radio.powerDown() cez radio_ui_handle_event).

Returns
true ak je rádio v stave ON, false ak je rádio vypnuté (powerDown).

Funkcia vracia vnútorný stav s_radio_on, ktorý sa mení najmä v rámci radio_ui_handle_event pri udalosti UI_BTN_DOWN_LONG.

Returns
true ak je rádio ON, false ak je vypnuté (po radio.powerDown()).
