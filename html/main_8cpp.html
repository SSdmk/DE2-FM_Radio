Detailed Description

Hlavný program FM rádia s enkóderom, tlačidlami a OLED displejom.

Aplikácia beží na ATmega328P pri 16 MHz a využíva:

  • Timer0 overflow interrupt na generovanie milisekundového čítača (timer_millis),
  • knižnicu uart.h na debug výpisy cez UART,
  • triedu Button na obsluhu štyroch tlačidiel,
  • triedu RotaryEncoder na čítanie rotačného enkódera (otáčanie + klik),
  • triedu Si4703 na ovládanie FM tunera,
  • funkcie z oled.h na vykresľovanie UI rádia na OLED displeji.

◆ main()

int main ( void )

Hlavná funkcia programu.

Postup:

  • inicializácia UART uart_init pre debug (9600 baud),
  • inicializácia OLED displeja oled_init,
  • nastavenie časovača 0 na overflow každú 1 ms a povolenie prerušení,
  • inicializácia tlačidiel Button::begin,
  • inicializácia enkódera RotaryEncoder::begin,
  • globálne povolenie prerušení sei,
  • inicializácia a nastavenie tunera Si4703:
    • Si4703::start,
    • Si4703::setChannel na 107.00 MHz,
    • Si4703::setVolume na hodnotu 10,
    • Si4703::powerDown a následne Si4703::powerUp,
  • nekonečná slučka:
    • čítanie udalostí z tlačidiel a enkódera,
    • mapovanie na UI udalosti cez radio_ui_handle_event,
    • debug výpis smeru enkódera cez UART,
    • čítanie aktuálneho stavu rádia (frekvencia, RSSI, hlasitosť, mute),
    • prekreslenie hlavnej obrazovky rádia na OLED pomocou oled_show_radio_screen.
Returns
V praxi nikdy nevracia, formálne 0.

◆ millis_counter

volatile unsigned long millis_counter = 0

Globálny čítač milisekúnd, inkrementovaný v prerušení TIMER0_OVF.

Každé pretečenie 8-bitového časovača 0 (pri nastavenom preskaleri) zodpovedá ~1 ms. Premenná sa používa v timer_millis a v debounce logike enkódera a tlačidiel.
